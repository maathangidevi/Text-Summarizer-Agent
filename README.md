# Text Summarizer Agent

This project is a simple AI-based Text Summarizer Agent developed using Python and TextBlob.

## Features
- Summarizes long text
- Easy to use
- Beginner friendly

## Technologies Used
- Python
- TextBlob
- NLP

## How to Run
1. Install Python
2. Install TextBlob:
   pip install textblob
3. Run:
   python text_summarizer.py

## Output
The program accepts user input text and displays a summarized version.
